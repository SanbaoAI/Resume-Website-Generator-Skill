理念

Skill 即产品

把一次性的 Prompt 变成可版本化、可分发、可迭代的开源资产。

01

Artifact 优先

每个阶段产出明确文件——JSON、Markdown、布局规格——而非直接跳到代码。

02

Agent 无关

Cursor、Claude Code、Codex 均可执行。Skill 是工作流，不绑定单一平台。

03

开源分发

一键 install.sh 安装，README 即文档，GitHub 即分发渠道。
